About

The Basic Local Alignment Search Tool (BLAST) finds regions of local similarity between sequences. The program compares nucleotide or protein sequences to sequence databases and calculates the statistical significance of matches. BLAST can be used to infer functional and evolutionary relationships between sequences as well as help identify members of gene families.

This is an open-source version of NCBI BLAST. You can find more information about BLAST at https://blast.ncbi.nlm.nih.gov/

SequenceServer

SequenceServer lets you rapidly set up a BLAST+ server for individual use or for sharing datasets with colleagues and community. Its minimalist, modern and thoughtful design lets you focus on the Biology and getting things done.

SequenceServer is free and open source. You can find more information about SequenceServer at http://www.sequenceserver.com/
